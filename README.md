# Elementals

Este repositorio está enfocado a practicar conceptos básicos de hacking. Todos los laboratorios podrán desplegarse por Docker.

## Footprinting & Networking

Para evaluar conceptos de subneting y escaneo de servicios, contamos con la aplicación `StringLock`. Este laboratorio consistirá en encontrar un puerto, en un host, que recibirá una cadena, si esta cadena es correcta devolverá un mensaje de respuesta, de lo contrario no devolverá nada. Esta aplicación puede ser configurada tanto para TCP como para UDP.

## System Security & Hardening

Para evaluar conexiones seguras contamos con la aplicación `SSHield`. Con este laboratorio se espera evaluar conceptos básicos de SSH y funcionamiento de servidores, así como navegación dentro de estos. La resolución de la máquina consistirá en establecer una conexión por SSH a un host y dentro de este, deberá encontrar una flag.

## Laboratorios

### Footprinting & Networking

**Objetivos de Aprendizaje**:
- Comprender y aplicar técnicas de footprinting para recopilar información sobre sistemas objetivo.
- Practicar subneting y escaneo de puertos para identificar servicios vulnerables.

**Descripción**:
El laboratorio `StringLock` te desafía a descubrir un puerto oculto en un host. Utilizando tus habilidades de escaneo de red, deberás encontrar el puerto correcto que está esperando recibir una cadena. Si la cadena es correcta, el sistema responderá con un mensaje clave; de lo contrario, permanecerá en silencio. Este laboratorio puede ser configurado para trabajar tanto en TCP como en UDP, ofreciendo así una experiencia de aprendizaje versátil que te permitirá familiarizarte con ambos protocolos de red.

### System Security & Hardening

**Objetivos de Aprendizaje**:
- Establecer y configurar conexiones SSH seguras.
- Navegar y explorar servidores para identificar y recopilar datos sensibles.

**Descripción**:
Con el laboratorio `SSHield`, explorarás los fundamentos de SSH y el funcionamiento de los servidores bajo esta capa de seguridad. Se espera que establezcas una conexión SSH segura a un host designado y, una vez dentro, te sumergirás en un entorno desafiante donde deberás encontrar una "flag" oculta. Este laboratorio no solo pondrá a prueba tus habilidades de conexión segura sino que también te enseñará técnicas de exploración y recolección de información dentro de sistemas comprometidos.
